# Studio Culinar-E

**Engineer Bold Compositions and Spread Culture.**

A flavor architecture tool by [Studio Monk-E](https://github.com/StudioMonk-E).

---

## What is this?

Studio Culinar-E treats ingredients as building materials. Instead of asking "what goes with what," it asks a harder question: *what connects two things that don't obviously belong together?*

Pick two ingredients. The studio drafts a **blueprint** — finding **keystones** (bridge ingredients) that structurally link them, ranked by bond strength. It also tells you when two ingredients already share a direct **foundation**, so you know when a bridge isn't even necessary.

109 ingredients. Symmetric pairing data. Cultural range from miso to mandioca to maple syrup.

## How to use

**The Blueprint** — Select two materials, choose a flavor palette, and draft a blueprint. The studio returns keystones ranked from structural (strong, iconic) to speculative (experimental, uncharted).

**Pantry Archive** — Explore a single ingredient: its type, taste profile, and every known connection, sorted by bond strength.

## Run it locally

```bash
pip install -r requirements.txt
streamlit run flavor_bridge.py
```

## Part of Studio Monk-E

Studio Culinar-E is one project inside a larger practice. Studio Monk-E is a cultural material lab — researching how ingredients encode history, labor, migration, and class. The tools we build reflect that framework.
